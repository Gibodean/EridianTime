# EridianTime
Vibe-coded with Gemini.

Amaze! Timekeeping for Eridians Celebrate Andy Weir’s Project Hail Mary with a dynamic watchface that calculates the current time in pure Base-6.

How to Read It:
Just like Rocky, you decode the numbers by counting the uninterrupted straight line segments in each glyph:

ℓ (Smooth loop) = 0
I (Single line) = 1
V (Two intersecting lines) = 2
λ (Main line split by a leg) = 3
+ (Two lines crossing) = 4
∀ (A 'V' split by a crossbar) = 5

The Layout:
Top Row: Hours (2 digits)
Middle Row: Minutes (3 digits)
Bottom Row: Seconds (3 digits)
(Example: A human minute of 59 is translated to base-6 as 135, displaying as I λ ∀)

Features:
Fully customisable colours for both Human and Eridian text. Want full immersion? Toggle off the Earth time and date in the settings, and the Eridian glyphs will dynamically scale up to fill the entire screen!
